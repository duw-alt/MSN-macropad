# MSN-macropad
MSN shaped macropad!!

![MSN macropad overall](https://github.com/duw-alt/MSN-macropad/blob/main/assets/macropad.png)


### Inspiration

I wanted to create a macropad that had a fun shape to sit at my desk. I created a macropad using 3 switches, 1 rp2040, 1 OLED and 1 rotary encoder. 

### Challenges

This was super challenging for me since it was my first time designing a schematic, pcb and a case. I stuggled a lot but it's DONE

### Specifications

BOM: 
- 3x Cherry MX Switches
- 3x Blank DSA Keycaps
- 1x XIAO RP2040
- 2x M3x16 Bolt
- 2x M3 Heatset
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x Case (2 printed parts)
